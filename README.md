# Call Center Analytics Dashboard

## Project Overview

This project analyzes call center operations using Power BI to track customer satisfaction, call volumes, agent performance, and operational efficiency.

## Tools Used

* Power BI
* Excel
* DAX

## Key Metrics

* Total Calls
* Answered Calls
* Resolved Calls
* Customer Satisfaction Score
* Average Call Duration
* Agent Performance

## Skills Demonstrated

* Data Cleaning
* Data Modeling
* Dashboard Design
* KPI Analysis
* Business Intelligence
